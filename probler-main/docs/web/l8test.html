Quick Start Guide

Prerequisites

  • Go 1.23.8 or later
  • Access to Layer 8 ecosystem repositories
  • Git for version control
  • Linux, macOS, or Windows with WSL2

Installation

# Clone the repository $ git clone https://github.com/saichler/l8test.git $ cd l8test/go # Run the automated test suite $ ./test.sh # Or run tests manually with coverage $ go mod init $ GOPROXY=direct GOPRIVATE=github.com go mod tidy $ go mod vendor $ go test -tags=unit -v -coverpkg=./infra/... -coverprofile=cover.html ./...

Quality Assurance Components

Service Handlers

TestServiceHandler

Basic service operations testing without transaction overhead

TestServiceTransactionHandler

Full transaction support with ACID compliance verification

TestServiceReplicationHandler

Distributed replication and cache consistency testing

Configuration Parameters

  • Node Count: Configurable test nodes (default: 4)
  • Port Ranges: Service ports [20000, 30000, 40000]
  • Log Levels: Trace, Debug, Info, Warn, Error
  • Replication Factor: Configurable replicas (default: 2)
  • Timeout Settings: Customizable per-operation timeouts

Example Test Implementation

package tests import ( "testing" "github.com/saichler/l8test/go/infra/t_topology" "github.com/saichler/l8test/go/infra/t_service" ) func TestDistributedService(t *testing.T) { // Create test topology with 4 nodes topology := t_topology.NewTestTopology(4) defer topology.Cleanup() // Initialize service with transaction support service := t_service.NewTestServiceTransactionHandler() service.Start(topology.GetNodes()) // Run distributed transaction test result := service.TestDistributedTransaction() if !result.Success { t.Errorf("Distributed transaction failed: %v", result.Error) } }
