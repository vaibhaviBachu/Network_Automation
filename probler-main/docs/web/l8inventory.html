Installation & Usage

Prerequisites

  • Go 1.25.4 or later
  • Layer 8 ecosystem dependencies

Quick Start

cd go/ ./test.sh # Manual setup go mod init GOPROXY=direct GOPRIVATE=github.com go mod tidy go mod vendor go test -tags=unit -v -coverpkg=./inv/... -coverprofile=cover.html ./...

Basic Usage

import ( inventory "github.com/saichler/l8inventory/go/inv/service" ) // Define your data types elemType := &YourProtobufMessage{} elemTypeList := &YourProtobufMessageList{} primaryKey := "Id" // Activate inventory service inventory.Activate( "links-id", // Links ID elemType, // Service item type elemTypeList, // Service item list type vnic, // Virtual NIC primaryKey, // Primary key field )

Supported Operations

POST

Add new inventory items to the distributed cache with automatic validation and indexing.

PUT

Replace existing inventory items with full object replacement and notification support.

PATCH

Update existing inventory items with delta changes and real-time notifications.

GET

Query and retrieve inventory data using SQL-like syntax with advanced filtering.

DELETE

Remove inventory items with proper cleanup and cascade operations support.
