What's New in v1.5.0

Enhanced security, local communication, and advanced service management

Two-Factor Authentication

Complete TFA support with TFASetup() and TFAVerify() methods. Includes Captcha protection and secure user registration workflows.

Local Communication

Same-VNic messaging optimization with Local() and LocalRequest() methods for reduced latency between co-located services.

Advanced SLA Decorators

New decorators for flexible service management: SetNonUniqueKeys() for flexible indexing and SetAlwaysOverwrite() for conflict resolution.

Powerful Features

Everything you need for building distributed systems at scale

Virtual Network Interface

Advanced VNic abstraction with multiple communication patterns: Unicast, Multicast, Round-robin, Proximity-based routing, and Leader selection for optimal message distribution.

Token Authentication

Secure token-based authentication system with validation, activation mechanisms, and comprehensive error handling for robust security operations.

Service Discovery

Built-in service registration and discovery with area-based routing, health monitoring, and intelligent endpoint selection for resilient systems.

Health Monitoring

Real-time system health tracking with Unix top-style output, CPU/memory usage statistics, message flow metrics, and comprehensive logging.

Transaction Management

Distributed transaction support with state tracking, commit/rollback operations, error handling, and comprehensive transaction lifecycle management.

Protocol Buffers

Comprehensive type definitions with multi-language support, efficient serialization, cross-platform compatibility, and extensible schema system.

System Architecture

Layered approach for maximum flexibility and performance

Application Layer - User Applications & Services
Service Layer - Business Logic with Service Areas
VNet Protocol - Message Routing & Service Discovery
Physical Network - TCP/UDP Transport Layer

Message Creation

Applications create messages with service targets using type-safe Protocol Buffer definitions

Intelligent Routing

VNic determines optimal destination based on service topology and communication patterns

Secure Transport

Messages are serialized, encrypted with AES, and sent via network protocols with authentication

Processing & Response

Receiving VNic deserializes, validates, and routes to appropriate handlers with full transaction support

Code Examples

See how easy it is to build distributed systems

Two-Factor Authentication (v1.5.0) 
// Setup TFA for a user
qrCode, secret, err := securityProvider.TFASetup(
    userId, vnic,
)
if err != nil {
    log.Fatal("TFA setup failed:", err)
}
// Display QR code for authenticator app

// Verify TFA code during login
err = securityProvider.TFAVerify(
    userId, totpCode, sessionId, vnic,
)

// Get captcha for bot protection
captchaImage := securityProvider.Captcha()

// Register new user securely
err = securityProvider.Register(
    email, password, captchaResponse, vnic,
)

// Fetch credentials securely
user, pass, token, session, err :=
    securityProvider.Credential(key, scope, resources)
Local Communication & Callbacks (v1.5.0) 
// Local communication for co-located services
err := vnic.Local(
    "local-service", 1, ifs.POST, data,
)

// Local request with response
response := vnic.LocalRequest(
    "cache-service", 1, ifs.GET, query, timeout,
)

// Service callbacks with continuation
type MyCallback struct{}

func (c *MyCallback) Before(
    data interface{}, action ifs.Action,
    isLocal bool, vnic ifs.IVNic,
) (interface{}, bool, error) {
    // Return: modified data, continue flag, error
    return preProcess(data), true, nil
}

Quick Start Guide

Get up and running in minutes with Layer 8 Types v1.5.0

1. Clone Repository

git clone https://github.com/saichler/l8types
cd l8types

2. Install Dependencies

cd go
go mod download

3. Generate Bindings

cd proto
./make-bindings.sh

4. Run Tests

cd go
./test.sh  # Automated tests with coverage
# Or manually:
go test -v -cover ./...

5. Build Your App

import "github.com/saichler/l8types/go/ifs"

// v1.5.0 - Local communication
vnic.Local("service", 1, ifs.POST, data)
// Two-Factor Auth
securityProvider.TFASetup(userId, vnic)
// Advanced SLA decorators
sla.SetNonUniqueKeys("category")
sla.SetAlwaysOverwrite("timestamp")

6. Deploy

docker build -t myapp .
docker run myapp
